# AutoML介绍

* [返回上层目录](../auto-machine-learning.md)



AutoML基本分以下几个方向：

* 自动数据清理（Auto Clean）
* 自动特征工程（AutoFE）
* 超参数优化（HPO）
* 元学习（meta learning） 
* 神经网络架构搜索（[NAS](https://flashgene.com/archives/tag/nas)）
